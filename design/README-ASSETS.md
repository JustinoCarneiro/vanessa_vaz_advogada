# Assets de Identidade Visual — Vanessa Vaz Marschallinger

## Estrutura

```
design/
├── logo/
│   ├── logo-dourado.png/.jpg/.pdf   → Logo horizontal, versão dourada (fundo branco/claro)
│   └── logo-teal.png/.jpg/.pdf      → Logo horizontal, versão teal (fundo neutro)
│
├── simbolo/
│   ├── simbolo-dourado.png          → Símbolo VM — versão dourada (favicon, ícone)
│   ├── simbolo-preto.png            → Símbolo VM — versão preta (impressão mono)
│   ├── simbolo-menta.png            → Símbolo VM — versão menta (fundos neutros)
│   ├── simbolo-teal-medio.png       → Símbolo VM — versão teal médio
│   └── simbolo-teal-escuro.png      → Símbolo VM — versão teal escuro
│
├── marcadagua/
│   ├── marcadagua-preto.png         → Marca d'água circular — preta
│   ├── marcadagua-menta.png         → Marca d'água circular — menta
│   ├── marcadagua-teal-medio.png    → Marca d'água circular — teal médio
│   ├── marcadagua-teal-escuro.png   → Marca d'água circular — teal escuro
│   └── marcadagua-branco.png        → Marca d'água circular — branca (fundo escuro)
│
└── manual/
    └── MANUAL_VANESSA_VEZ.pdf       → Manual completo de identidade visual
```

## Regras de uso rápido

| Contexto | Logo | Símbolo |
|---|---|---|
| Header fundo branco | logo-dourado | simbolo-dourado |
| Header fundo teal escuro | logo branco (no manual) | simbolo-branco |
| Header fundo menta | logo-teal | simbolo-teal-escuro |
| Favicon / ícone app | — | simbolo-dourado |
| Rodapé fundo escuro | logo branco | marcadagua-branco |
| Marca d'água em PDF | — | marcadagua-teal-medio |

## Paleta oficial

| Nome | HEX | RGB |
|---|---|---|
| Teal escuro | `#3D5C5F` | 61, 92, 95 |
| Teal médio | `#89B0AF` | 137, 176, 175 |
| Menta | `#BDE4DA` | 189, 228, 218 |
| Preto | `#000000` | 0, 0, 0 |
| Branco | `#FFFFFF` | 255, 255, 255 |

> Dourado: exclusivo para o logo. Não usar em UI.

## Tipografia oficial
- **Display:** Petita (fallback: Cormorant Garamond)
- **Corpo / UI:** Montserrat
